# Insider Threat Detection Playbook

This repository contains a Python-based insider threat detection prototype and a comprehensive incident response playbook for financial services institutions. The goal is to showcase how to detect high-signal indicators of insider risk, enrich alerts with HR context and triage them, and document legal and HR processes.

## Components

- **insider_threat_detection.py** – Python script that ingests log events and HR context and produces triage reports with risk scores.
- **sample_events.json** – Example log events used for testing the script.
- **hr_lookup.json** – HR context used to enrich alerts in the script.
- **triage_reports.json** – Example output generated by running the script on the sample data.
- **insider_threat_playbook.md** – Incident response playbook covering detection logic, risk scoring and IR procedures with HR and legal.

## Running the script

1. Clone this repository.
2. Install Python 3 (if not already installed).
3. Run the detection script from the repository root:

```bash
python insider_threat_detection.py
```

The script reads `sample_events.json` and `hr_lookup.json`, applies detection rules (bulk downloads, unauthorized resource access, and after‑hours queries), enriches with HR context, and writes a list of triage reports to `triage_reports.json`. It also prints the triage reports to stdout.

The playbook (`insider_threat_playbook.md`) explains the detection logic, risk scoring, and full incident response workflow, including legal hold and HR coordination.
